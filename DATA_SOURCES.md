## 2. 植保知识库

### 2.1 作物识别图谱（crop_atlas，18 种）
覆盖粮食（水稻、小麦、玉米、马铃薯）、经济作物（棉花、油菜、花生、大豆、甘蔗）、果树（柑橘、苹果、葡萄、香蕉、芒果）、蔬菜（番茄、辣椒）、茶叶、烟草。
- **生育期、主要病虫害**：依据《中国农作物病虫害》、全国农技推广中心病虫情报常识整理。
- **飞防要点**（飞行高度、亩喷液量、作业时段、关键防治期）：依据植保无人机飞防作业通用农技规范整理。

### 2.2 农药配比库（pesticide_ratio，30 条）
- **农药品种与剂型**（如 25%吡蚜酮悬浮剂、5%甲氨基阿维菌素苯甲酸盐水分散粒剂、430g/L 戊唑醇悬浮剂、井冈霉素A、Bt、20%三环唑等）：依据**中国农药信息网农药登记数据库**中真实登记产品整理。
- **亩用药量、亩喷液量、安全间隔期**：依据农药标签推荐剂量与飞防减量稀释经验值整理（飞防通常较常规喷雾减少用水量，需添加飞防助剂）。
- **注意事项**：依据抗性治理（农药轮换）、二次稀释、避开花期/高温等农技安全用药准则。

> 说明：农药剂量仅作培训参考，实际作业须以**当地植保部门指导**与**农药标签**为准。

## 3. 禁飞区 / 限飞区（no_fly_zone，16 条）
- 以**全国主要民用机场净空保护区**为主（首都、大兴、浦东、虹桥、白云、宝安、双流、江北、咸阳、萧山、天河、新郑、珠海金湾等）+ 城市重点区域 + 限飞区示例。
- **经纬度采用高德地图 GCJ-02 坐标系真实坐标**，在前端高德地图上以圆形净空范围渲染。
- 净空半径依据机场净空保护一般范围（约 6–9 km）设定。
- 地图能力由**高德地图 Web 端 JS API 2.0** 提供（见下）。

## 4. 题库与试卷（question 40 / exam 5）
- 题目围绕**法规、安全、植保技术、飞行操作、气象、设备维护**六个维度命制，知识点依据上述法规与农技资料。
- 试卷总分与所引用题目分值之和**严格一致**（已通过 SQL 校验）。
